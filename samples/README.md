# Demo Media

Place your source video files here for testing ShortsForge.

## Required files for the demo

| File | Description | Source |
|------|-------------|--------|
| `podcast.mp4` | A 5–30 minute podcast or interview | Your own recording |
| `azure-overview.pdf` | Azure documentation for Foundry IQ KB demo | [docs.microsoft.com](https://docs.microsoft.com/azure) |

## Media Provenance Requirements

- Only use media you own or have rights to
- CC0 and CC-BY licensed content is acceptable for demo purposes
- Never commit copyrighted media without explicit license
